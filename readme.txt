******УСТАНОВКА******

При установке приложения, необходимо разрешить доступ к файловой системе, контактам, камере.

******ЗАПУСК******

При запуске приложения, будет отображен список картинок с подписями. По умолчанию уже доступно две картинки. 

******РЕДАКТИРОВАНИЕ ЭЛЕМЕНТА******

Для редактирования необходимо кликнуть на элементу картинки.
Далее будет отображено окно с картинкой и полями ввода для названия и комментария к картинке. Изменение названия внутри приложения не влечет изменения названия файла в системе. Так же доступны кнопки сохранения изменений и отсылки картинки по почте.

******ОТСЫЛКА ПОЧТЫ******

При нажатии на кнопку отсылки почты, будет открыто окно для выбора адреса получателя. Для выбора адреса необходимо кликнуть по поллю ввода адреса. После чего в списке контактов необходимо выбрать адресата. По умолчанию в поле поставляется первый адрес электронной почты контакта. Далее, по нажатию кнопки отправки, необходимо выбрать почтовый клиент, через который будет происходить отправка.


******ДОБАВЛЕНИЕ КАРТИНОК******

Для добавления новой картинки, необходимо нажать на соответствующую кнопку. Будет отображено диалоговое окно, в котором необходимо выбрать, добавить картинку из фаловой системы или создать новую, при помощи камеры.
Если выбран вариант создания картинки через камеру, передается управлене камере. После создания картинки, приложение предложит выбрать созданную картинку на карте памяти.
